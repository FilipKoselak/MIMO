Using other functions of Mimo

  1. Leaderboard
    1. To access the leaderboard click on the "cup" icon below.
    2. Now you can see your league and your rank.
  2. Community
    1. Click on "Community" icon below to acces the forum.
    2. Now you can search for any topic on a community forum.
  3. Mimo Store
    1. Click on the "coin" icon to access Mimo Store.
    2. Now you can see the Mimo Store.
  4. Activity
    1. Click on the "flame" icon to acces Activity Calendar.
    2. Now you can see the Activity Calendar.
  5. Playground
    1. Click on the "profile" tab to access playgrounds.
    2. Click on "CREATE" button to create a playground.
    3. Select a template from: Python program, JavaScript program, Simple website and React app.
    4. You can code anything you want in the playground.
